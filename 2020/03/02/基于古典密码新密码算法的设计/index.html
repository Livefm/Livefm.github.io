基于古典密码新密码算法的设计

序章

​ 随着大三上密码学这一门课程结束，对密码学有了个更直观的感受。但是课本上的知识过于理论化，为了更好的消化其中的知识，于是想着自己动手设计一款基于古典密码的新密码算法。并且用编程语言去实现这个算法。

Substitution_Cipher.py

import random
import VigenereEncrypto  as v1
Substitution_Key = dict(a = 'i',b = 'd',c = 'j',d = 'k',e = 'e',
                        f = 'u',g = 'm',h = 'v',i = 'c',j = 'r',
                        k = 'l',l = 'f',m = 's',n = 't',o = 'b',
                        p = 'w',q = 'q',r = 'z',s = 'g',t = 'o'
                        ,u = 'p',v = 'h',w = 'a',x = 'y',y = 'n',z = 'x')
def SubstitutionEncryp(plaintext):#代换密码加密
    ciphertext = ""
    for each in plaintext:
        ciphertext = ciphertext+(Substitution_Key[each])
    k1 = random.randint(97,122)
    k2 = random.randint(97,122)
    k3 = random.randint(97,122)
    k4 = random.randint(97,122)
    secret_key = chr(k1)+chr(k2)+chr(k3)+chr(k4)#随机生成四位维吉尼亚密钥
    temp = v1.VigenereEncrypto(ciphertext,secret_key)
    return temp
def SubstitutionDecryp(ciphertext):#代换密码解密
    plaintext = ""
    for each in ciphertext:
        for (key,value) in Substitution_Key.items():
            if each == value:
                plaintext += key
    return plaintext

VigenereDecrypto.py

import Substitution_Cipher as s
def VegenereDecrypto(ciphertext):#维吉尼亚密码解密
    ciphertext_temp = ciphertext[:len(ciphertext)-4]
    secret_key = ciphertext[len(ciphertext)-4:]
    plaintext = ""
    for i in range(len(ciphertext_temp)):
        j = i%len(secret_key)
        temp = (ord(ciphertext_temp[i])-ord(secret_key[j]))%26
        if temp>0:
            plaintext += str(chr(temp+97))
        else:
            plaintext += str(chr(temp+26+97))
    temp1 = s.SubstitutionDecryp(plaintext)
    return temp1

VigenereEncrypto.py

def VigenereEncrypto(plaintext,secret_key):#维吉尼亚密码加密
    plaintext_len = len(plaintext)
    key_len = len(secret_key)
    cipher = ""
    for i in range(plaintext_len):
        j = i%key_len
        cipher += str(chr(((ord(plaintext[i])+ord(secret_key[j])-97*2)%26+97)))
    return cipher+secret_key

加密过程解释：

首先将输入的明文用代换密码进行加密，然后再用random模块随机生成四个97到122的整数，将这四个整数用chr()函数转换成四个英文字母构成一个四位的密钥，然后利用生成的随机密钥，将经过代换加密的明文再次用维吉尼亚密码加密。最后输出的密文是密文加上随机密钥的组合。以便解密

解密过程解释：

解密就是加密的逆过程，先将密文里面的真正的密文和密钥分开，然后通过维吉尼亚的解密函数得到代换密码的密文，然后将这个密文传到代换密码解密函数，得到真正的明文。

总结

整个密码体系只是利用到了古典密码算法里面代换和维吉尼亚两种思想，但是通过将两者有机结合，使得加解密变得复杂一下，再加上密钥是通过随机数生成函数random生成，让密钥不是一成不变。但是整套体系其实也有不足，如：如果有第三方知道了密钥就隐藏在明文中，整个加密过程就变得完全透明不安全。这也是后期需要改进的地方。
